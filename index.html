Internet Explorer 后续安装

此页面提供了 Windows Internet Explorer 的后续设置选项。

搜索提供程序

搜索提供程序是支持 Internet Explorer 的搜索建议和地址栏中的自动搜索的加载项。您可以安装多个搜索提供程序，但只有一个是默认搜索提供程序。

地址栏中的搜索建议可以在你键入时提供相关的搜索词，从而帮助你优化搜索。这些建议来自您的默认搜索提供程序。若要打开或关闭搜索建议，请打开 Internet Explorer，然后开始在地址栏中输入某个搜索词，然后在下拉菜单中选择“打开建议 (或者关闭建议)”。

至少需要 Internet Explorer 7 才能安装搜索提供程序。如果您是 Windows XP Service Pack 2 用户，请安装 Internet Explorer 8。

要安装搜索提供程序，请单击下方的搜索引擎，然后单击“添加”。

跟踪保护列表 (Tracking Protection Lists)

跟踪保护有助于防止关于你浏览的信息发送到所访问站点上的第三方内容提供商。可以将跟踪保护列表视为“不要呼叫”列表。Internet Explorer 会阻止来自列表中站点的任何第三方内容，并限制这些第三方站点可能收集的有关你的信息。

至少需要 Internet Explorer 9 才能使用跟踪保护列表。如果您是 Windows Vista 或者 Windows Server 2008 用户，请 从 Microsoft Download Center 下载 Internet Explorer 9。如果您是 Windows 7 Service Pack 1 或者 Windows Server 2008 R2 Service Pack 1 用户，请 下载 Internet Explorer 11 (脱机安装程序)。

要安装跟踪保护列表，请单击下方的跟踪保护列表提供商，然后单击“添加列表”。
